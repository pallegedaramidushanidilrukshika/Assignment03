Shape Area and Perimeter Calculator (Java)

Description

This project is a simple Java program that demonstrates Object-Oriented Programming (OOP) concepts. It calculates the area and perimeter of different shapes such as Circle, Rectangle, and Square.

The program uses inheritance and abstract classes to represent different shapes and their behaviors.

Features

* Calculate area and perimeter of a Circle
* Calculate area and perimeter of a Rectangle
* Calculate area and perimeter of a Square
* Demonstrates OOP concepts such as abstraction, inheritance, and method overriding

OOP Concepts Used

* Abstract Class
* Inheritance
* Encapsulation
* Method Overriding

Technologies Used
Java

How It Works

1. The abstract class `Shape` defines common properties such as color and filled status.
2. The classes `Circle`, `Rectangle`, and `Square` extend the Shape class.
3. Each class implements methods to calculate area and perimeter.
4. The main class creates objects of each shape and displays the results.
